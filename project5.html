Git

Le projet git à pour but de nous familiariser avec cet incroyable outils qu'est Git. Non seulement il nous sera d'une utilité hors norme, il permet aussi à nos fabuleux proffesseurs de noter nos projets. Il est indéniable que Git est un outils dont la maitrise est nécessaire pour pouvoir travailler de manière professionelle avec plusieurs dévelloppeurs.

Voici un screenshot de git bash, le premier outils du parfait dévelloppeur. Il permet d'enregistrer chaque modification et de le uploader soit sur gitHub, soit de les partager avec ses collaborateurs. De plus, le système de commit permet de faire revenir sur une version antérieure du projet (au cas ou quelques mésaventure se produisent).

Voici gitHub, la gentille contribution de Microsoft à git. Un site qui permet d'host et de partager vos projets. Bien sûr, si vous voulez des projet privés, ce sera au moyen de quelques sous, mais pour l'instant, je ne crois pas en avoir l'utilité. C'est via ce site que nos proffesseurs téléchargent et analysent nos projets.

Grâce à ce fabuleux site, nous pouvons non seuelement suivre l'avancement de nos projets mais aussi avoir des données sur le nombres de modifications apportées, par qui et quand. C'est un outils bien utile pour les codeurs comme les employeurs, car il fait bien souvent office de portfolio. On peut peut-être, pour une fois , bien que ça fasse mal au coeur, dire merci à microsoft pour ça.
