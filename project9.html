Connexion page

Le projet connexion est le 3ème projet que j'ai fait durant la projectPool n°1. Le concept est simple: Faire une page HTML pour se connecter, via laquelle il est possible d'accéder à la page d'inscription ou à la page de mot de passe oublié. La page d'inscription doit posséder certains champs tels que nom, numéro de téléphone ou adresse mail et doit aussi vérifier la présence/ le type de donnée entrée par l'utilisateur.
